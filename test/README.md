# Test and archive

Thư mục này không được notebook sử dụng khi chạy trên Colab.

- `archive/`: pipeline cũ, tài liệu, reports, stage notes, test source và maintenance scripts.
- `runtime_evidence/`: output JSON, diagnostics và dữ liệu tạm dùng khi kiểm thử; bị Git ignore.

Các thành phần runtime thật được giữ ở thư mục gốc: `clinical_nlp_lab/`,
`artifacts/`, `tools/`, notebook và `requirements-colab.txt`.
